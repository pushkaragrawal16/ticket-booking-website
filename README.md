This is a Movie-ticket-booking website which is built using React.js, Node.js, Express.js and MongoDB. It has the following features:
1. User can book movie tickets online.
2. User can sign up through Google login.
3. Admin can add, update and delete movies.
4. User can view the list of booked tickets.
5. User can view movie details.
6. Responsive design for mobile and desktop users.

## Technologies Used

1. React.js
2. Node.js
3. Express.js
4. MongoDB
5. HTML
6. CSS
7. JavaScript

Only home page movies are dynamically connected to the database and have seatmatrix for each movie.
user can book tickets for a movie and the seat matrix will be updated in the database.
Movies in Navbar is active and will display movies according to users choice.However kind of movies you will be routed through Movies in Navbar will show you static pages till now.

## Installation

1. Clone the repository.
2. Install dependencies using `npm install`.
3. Run the server using `npm start`.
4. In Terminal: `cd backend`
5. In Terminal: `nodemon server.js`
6. open http://localhost:3000 in your browser.
